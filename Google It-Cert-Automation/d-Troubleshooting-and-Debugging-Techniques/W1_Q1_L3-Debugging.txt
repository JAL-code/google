Definitions: 
troubleshooting = process of identifying, analyzing, and solving problems (system)
debugging = process of identifying, analyzing, and removing bugs in a system. (code)

tcpdump/Wireshark - Ongoing network connections and cable traffic.
ps/top/free - # and types of resources used in the system
strace = system calls made by a program
ltrace = library calls made by the software.

Debuggers let us follow the code line by line, inspect changes in variable assignments, interrupt the program when a specific condition is met, and more.
